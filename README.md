# iStarken - Cybersecurity Blog

This repository contains the source code for my cybersecurity blog, hosted at [istaarkk.github.io](https://istaarkk.github.io/).

## About

This site is built with Jekyll and hosted on GitHub Pages. It contains articles and resources about various cybersecurity topics including:

- Web Security
- Low Level/Binary Exploitation
- Network Security
- Reverse Engineering
- Satellite Communications
- And more!

## Structure

- `_posts/`: Contains all the blog articles
- `assets/`: Contains images and other static assets
- `_config.yml`: Jekyll configuration
- `.nojekyll`: Ensures GitHub Pages doesn't process the site with Jekyll (when we want to use our own build process)

## Local Development

To run this site locally:

1. Install Jekyll and Bundler: `gem install jekyll bundler`
2. Clone this repository: `git clone https://github.com/Istaarkk/istaarkk.github.io.git`
3. Navigate to the repository: `cd istaarkk.github.io`
4. Start the Jekyll server: `jekyll serve`
5. View the site at: `http://localhost:4000`
